# Credit Card Transaction Analytics Dashboard (Power BI)

## Project Overview

This project presents an interactive Power BI dashboard analyzing credit card transactions of state employees.

The dashboard provides insights into spending patterns, department-wise expenditure, merchant-level spending, and transaction trends.

## Tools Used

- Power BI Desktop
- Power Query
- DAX
- Data Modeling
- Data Visualization

## Dataset Features

The dataset includes:

- Department Name
- Merchant
- Category Description
- Transaction Date
- Fiscal Period
- Merchandise Amount

## Dashboard Pages

### Executive Overview
Shows key KPIs and monthly spending trends.

### Department Analysis
Analyzes department-wise spending and transaction patterns.

### Merchant & Category Analysis
Identifies top merchants and category spending distribution.

## Key Insights

- Total spending across departments exceeds $670M.
- Some departments account for a large percentage of transactions.
- Telecom and industrial supply categories dominate spending.
- Merchant-level analysis highlights major vendors.
